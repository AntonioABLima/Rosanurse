# Rosanurse

Plataforma de leitura de exames laboratoriais em PDF para médicos e profissionais de saúde.

## Sobre

Rosanurse é uma aplicação web **100% frontend** — sem backend. O processamento do PDF acontece diretamente no navegador via `pdfjs-dist`.

## Stack

- **Next.js 16** + TypeScript
- **Tailwind CSS v4**
- **pdfjs-dist** — parsing de PDF no browser

## Funcionalidades

- [x] Leitura de exames laboratoriais em PDF
- [ ] Anamnese *(em breve)*
- [ ] Consulta *(em breve)*

## Desenvolvimento

```bash
cd src/frontend
npm install
npm run dev
```

Acesse em `http://localhost:3000`.
